# Maharaja-Masala

A Next.js 14 website built with TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Production

```bash
npm run build
npm start
```

### Docker

```bash
docker compose up --build
```

The app runs on port 3000.
